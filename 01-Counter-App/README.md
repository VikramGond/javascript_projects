# Counter App

A simple and responsive counter application built with HTML, CSS, and JavaScript.

## 📸 Preview

![Counter App](assets/images/counter-preview.gif)

---

## ✨ Features

- Increment counter
- Decrement counter
- Reset counter
- Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository.
2. Open the `01-counter-app` folder.
3. Run `index.html` in your browser.
